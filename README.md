## 🧠 OPUS — Task Intelligence Dashboard

A powerful, productivity-focused Task Management Web App built using HTML, CSS, and JavaScript, featuring intelligent task tracking, smart insights, real-time countdowns, and a clean editorial UI for high-performance productivity.

## 🚀 Features
* **Advanced Task Management (CRUD)** – Create, edit, delete, and manage tasks with priority and deadlines.
* **Smart Dashboard Analytics** – Track completion rate, pending tasks, due today, and overdue tasks.
* **Priority-Based Task System** – Organize tasks into High, Medium, and Low priorities.
* **Real-Time Countdown System** – Displays live countdown timers for task deadlines.
* **Smart Insights Banner** – Shows contextual insights like overdue tasks and high-priority alerts.
* **Powerful Filtering System** – Filter tasks by pending, completed, high priority, and overdue.
* **Advanced Sorting Options** – Sort tasks by creation date, deadline, or priority levels.
* **Focus Mode** – Displays only high-priority tasks for distraction-free productivity.
* **Search Functionality** – Search tasks instantly by title or description.
* **Recurring Task System** – Supports daily and weekly recurring tasks automatically.
* **Drag & Drop Reordering** – Reorder tasks dynamically with smooth drag-and-drop interactions.
* **Notification Reminders** – Sends deadline reminders using browser notifications.
* **LocalStorage Persistence** – Saves all tasks and state locally in the browser.
* **Theme Toggle (Dark / Light)** – Switch between dark and light UI modes.
